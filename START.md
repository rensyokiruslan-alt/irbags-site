## Быстрый старт

### 1. Откройте страницу входа

```bash
# Если есть Python
python3 -m http.server 8000
# или с Python 2
python -m SimpleHTTPServer 8000
```

Потом откройте http://localhost:8000/admin/

### 2. Перейти на страницу входа

Просто откройте `admin/index.html` в браузере (если не нужен веб-сервер для локального развития).

### 3. Введите пароль

**Пароль:** `irbagsmd`

- Каждая буква отображается чёрной точкой (10×10px)
- При ошибке надпись «пароль» становится красной + лёгкий шейк
- Любой символ после ошибки убирает красный цвет
- Enter или клик на «войти» = отправка

### 4. После логина

Редирект на `dashboard.html` (заглушка). Там потом будет настоящая админка.

## Особенности

### Входная страница (макет Figma 272:900 / 272:2335 / 327:896)

- ✅ **Default** — «пароль» с opacity 0.2 (как плейсхолдер)
- ✅ **Entry** — точки 10×10 с `mix-blend-mode: exclusion`
- ✅ **Error** — красный текст `#ff4646` + шейк анимация
- ✅ **Пароль** — `irbagsmd` (легко, для демо)
- ✅ **Enter или клик** — отправка

### Цвета

```css
--color-white:     #ffffff
--color-black:     #000000
--color-gray-100:  #efefef  (основной серый)
--color-gray-200:  #e4e4e4
--color-gray-300:  #d7d7d7
--color-error:     #ff4646  (ошибки)
```

### Типографика

- **Основной шрифт:** Helvetica Neue Medium 500
- **Body:** 20px, -0.4px letter-spacing, lowercase
- **Lead:** 35px, -0.7px letter-spacing
- **Всё lowercase по умолчанию**

## Локальный развёртывание

### Вариант 1: простой веб-сервер

```bash
# Python 3.x
python3 -m http.server 8000

# Node.js (если установлен)
npx http-server

# Live Server в VS Code
# — установить расширение, потом кнопка "Go Live"
```

Откройте http://localhost:8000/admin/

### Вариант 2: прямое открытие в браузере

Просто откройте файл `admin/index.html` в браузере двойным кликом. Работает, но некоторые возможности могут быть ограничены (например, автозаполнение пароля).

## 🔒 Безопасность (важно!)

**Текущая реализация:** проверка пароля в JavaScript на клиенте.  
**⚠️ Это не безопасно для продакшена!** Любой может открыть DevTools и увидеть пароль в коде.

### Для продакшена нужно:

1. **Прятать админку за HTTP Basic Auth** на уровне веб-сервера (nginx/Apache)
2. **Или перенести проверку на сервер** (Node, Python, PHP — не важно)
3. **Или использовать правильную аутентификацию** (OAuth, JWT и т.д.)

Сейчас — это прототип для дизайна и локального тестирования.
